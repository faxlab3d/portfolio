Hong Kong Obscure

Hong Kong Obscure is a psychadelic and liminal desktop and VR game/experience I built using UE5. I utilized Nanite and Lumen extensively in this project. It features complex shaders and trippy visuals for a strange and wonderous atmosphere.

The environment was built procedurally inside Houdini and then ported to UE5 using Houdini for Unreal Engine plugin.

Using box meshes and simple planes I modeled the whole environment inside blender, used the blockout meshes inside Houdini with a custom built streets and buildings generators to create the whole environment.

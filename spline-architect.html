Spline Architect

Spline Architect is a plugin that I started making for myself to use for environments I create. It works in a non-destructive manner using Splines and modular meshes making it easy to change or update built environments.

Creating various custom meshes or full buildings inside other applications is time consuming and hard to modify post-factum.Â 

Spline Architect solves this issue by generated nacesarrry dynamic elements like corners and floors from modular meshes and by using Geometry Scripting library that comes with UE5.

It bakes generated structures into regular actors in-editor avoiding any runtime logic or construction scripts making levels load as fast as if meshes were places by hand.

Its a battle tested plugin and its availabe for purchase on FAB

GAZOLINAS

GAZOLINAS is an semi-realistic arcade racing simulator I built using UE4. It features arcady but still true to reality vehicle physics (using Blue Man Vehicle Physics plugin).

The cars can have their parameters configured for various scenarios. The game features online Steam Leaderboards with replays.

As with all my games I created the environments, 3d assets and code all by myself.
